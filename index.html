Overview: This project focuses on detecting fraudulent credit card transactions using a hybrid machine learning approach.

Problem: Online transactions are increasing, leading to a higher risk of fraud. Traditional rule-based systems are not effective.

Solution:

  • Machine Learning (Random Forest)
  • Behaviour Drift Analysis

Working:

  • ML predicts fraud probability
  • Behaviour drift checks deviation from normal usage
  • Final score is calculated using fusion (70% ML + 30% drift)

Features:

  • Detects fraud with higher accuracy
  • Reduces false positives
  • Adapts to user behavior

Technologies: Python, Scikit-learn, Pandas, Machine Learning

Outcome: Improved fraud detection accuracy compared to traditional methods
